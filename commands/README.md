# commands/ — skills GÉNÉRALISTES du cœur (partagés, gouvernés)

Les commandes ici sont **project-agnostiques** (aucun chemin en dur) et générées pour tout projet qui les
sélectionne. Une commande **project-spécifique** vit dans `.ai/commands/` (pas ici).

> **Aujourd'hui : aucune.** La **délibération** (proposeur ↔ réfuteur) n'est **PAS** une commande à
> invoquer — c'est un **skill de base**, toujours actif, décrit dans
> [`../conventions/method.md`](../conventions/method.md) et inliné dans chaque adapter. On n'invoque pas
> « réfléchis » : l'IA délibère **par défaut**. (Mettre la méthode dans une commande la rendrait *opt-in*,
> ce qui contredit « délibère AVANT de proposer ».)

Format d'une commande : `<nom>/command.md` (+ fragments `<stack>.md` additifs) — cf. `tools/sync-ai.md`.
